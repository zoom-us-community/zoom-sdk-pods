### About

This is cocoapods for Zoom IOS SDK. 

Pod includes all architectures.
(Device and Emulator)


Pod includes sdk list:
- MobileRTC.framework
- MobileRTCScreenShare.framework
- MobileRTCResources.bundle

### Usage
Update your Podfile:
```
pod 'ZoomSDK', '5.7.1.645'
```
